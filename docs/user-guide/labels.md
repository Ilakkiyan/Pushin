# Labels

Labels are a flat, cross-cutting taxonomy that can apply to tasks, events, habits, pages, and projects.

## What Labels Do

Labels help organize work across entity types. They can also carry scheduling preferences for tasks:

- preferred time window
- minimum block size
- maximum block size
- batching preference

## Calendar Filters

Week and month views can filter by label. They can also color items by primary label instead of project or event type.

## AI Suggestions

Pushin can suggest labels from deterministic keywords after AI planning. Suggestions are confirmed before being applied.
