# Tasks and Projects

Tasks are the units the scheduler can place on the calendar. Projects group related tasks.

## Task Fields

Tasks can carry:

- estimate
- deadline
- priority
- minimum and maximum chunk size
- dependencies
- status
- labels

## Scheduling

The scheduler places task blocks into free time around fixed events, routines, locked blocks, deadlines, and dependencies.

## Projects

Projects can be active or archived. Project colors provide the default calendar color for task blocks unless label coloring is enabled.
