# Notes and Vault

The vault is a local-first document space for notes, plans, daily pages, and knowledge.

## Pages

Create pages from the sidebar and write with the block editor. Pages can be nested into a tree.

## Wikilinks

Type `[[` to link to another page. Linked pages show backlinks under **Linked references**.

## Daily Notes

Each calendar day can have a page. Daily notes connect time and knowledge without leaving the app.

## Semantic Recall

When the embedding server is available, Pushin can search by meaning. If embeddings are unavailable, keyword search still works.

## Graph

The graph view shows pages as nodes and wikilinks as edges.
