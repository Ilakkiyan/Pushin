# Habits

Habits track recurring routines and consistency.

You can:

- create daily, weekly, or interval habits
- set a color and duration
- mark days complete
- backfill history from the heatmap
- add a habit to today's calendar

When a habit is scheduled, Pushin creates calendar time for it and plans tasks around that commitment.

## Drag to set your preferred time

By default Pushin best-fits each habit into an open slot. **Drag a habit block on the calendar** to
move it to the time you actually want — Pushin **learns that as your preferred time** and uses it when
scheduling future days, without disturbing occurrences you've already arranged. It's a soft preference:
if the slot is taken on a given day, Pushin falls back to the best open time.

**Double-click** a habit block to jump to the Habits page.
