# Calendar

The calendar has a full week grid and a month overview.

## Week

- Full 24-hour grid
- Drag task blocks to move them (which pins them in place); click the lock badge on a pinned block to unpin
- Double-click a task block to open its linked note
- Drag a **habit** block to set your preferred time — Pushin learns it (see [Habits](/user-guide/habits))
- Click empty space to add fixed busy time
- Reserved routines and sleep windows are shaded behind the grid

Adding or changing a task keeps your existing scheduled blocks where they are — Pushin slots the new work in around them instead of reshuffling the whole calendar.

## Month

Month view shows per-day chips for events and task blocks. Click a day to jump to its week.

## Daily Notes

Use the note button on a calendar day to open or create that day's vault page.

## Labels

Calendar views can color items by primary label and filter the grid by selected labels.
