# Getting Started

Pushin is a Tauri desktop app with a React frontend, Rust backend, local SQLite database, and optional local AI server.

## Fast Path

```bash
git clone https://github.com/Ilakkiyan/Pushin.git
cd Pushin
npm install
npm run tauri dev
```

On first launch, use the setup card in the chat panel to download a local model, or point Pushin at an existing Ollama server.
